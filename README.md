# Korrektunator [DOWNLOAD THE INSTALLER HERE](https://www.dropbox.com/s/a3lz7gliebv8a29/korrekturnator_setup.exe?dl=0)



## What can you do with it?

- Practice German declensions, verb conjugations and much more
- Uses [spaCy](https://github.com/explosion/spaCy) engine
- Works with any German text
- No coding required!
- Download,install and start
- Source code included!
- For Windows!
